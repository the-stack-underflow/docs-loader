Fields

buffer: [255]u8,

Contains the memory used to store the dynamic linker path. This field should not be used directly. See get and set. This field exists so that this API requires no allocator. 

max_byte: ?u8,

Used to construct the dynamic linker path. This field should not be used directly. See get and set.
