ensureTotalCapacityPrecise(
    self: *Self, 
    new_capacity: usize,
) Allocator.Error!void

Modify the array so that it can hold at least new_capacity items. Like ensureTotalCapacity, but the resulting capacity is much more likely (but not guaranteed) to be equal to new_capacity. Invalidates pointers if additional memory is needed.
