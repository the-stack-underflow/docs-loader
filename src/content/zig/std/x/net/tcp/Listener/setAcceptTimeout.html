setAcceptTimeout(
    self: Listener, 
    milliseconds: usize,
) !void

Set a timeout on the listener that is to occur if no new incoming connections come in after a specified number of milliseconds. A subsequent accept call to the listener will thereafter return error.WouldBlock should the timeout be exceeded.
