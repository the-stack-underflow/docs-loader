setReadTimeout(
    self: Client, 
    milliseconds: u32,
) !void

Set a timeout on the socket that is to occur if no messages are successfully read from its bound destination after a specified number of milliseconds. A subsequent read from the socket will thereafter return error.WouldBlock should the timeout be exceeded.
