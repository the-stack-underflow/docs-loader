readlinkat(
    dirfd: fd_t, 
    file_path: []const u8, 
    out_buffer: []u8,
) ReadLinkError![]u8

Similar to readlink except reads value of a symbolink link relative to dirfd directory handle. The return value is a slice of out_buffer from index 0. See also readlinkatWasi, realinkatZ and realinkatW.
