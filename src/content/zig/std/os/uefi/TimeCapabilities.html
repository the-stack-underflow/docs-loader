Capabilities of the clock device

Fields

resolution: u32,

Resolution in Hz 

accuracy: u32,

Accuracy in an error rate of 1e-6 parts per million. 

sets_to_zero: bool,

If true, a time set operation clears the device's time below the resolution level.
