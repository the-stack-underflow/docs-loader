strHash(
    password: []const u8, 
    options: HashOptions, 
    out: []u8,
) Error![]const u8

Compute a hash of a password using 2^rounds_log rounds of the bcrypt key stretching function. bcrypt is a computationally expensive and cache-hard function, explicitly designed to slow down exhaustive searches.

The function returns a string that includes all the parameters required for verification.

IMPORTANT: by design, bcrypt silently truncates passwords to 72 bytes. If this is an issue for your application, hash the password first using a function such as SHA-512, and then use the resulting hash as the password parameter for bcrypt.
