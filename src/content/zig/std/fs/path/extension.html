extension(
    path: []const u8,
) []const u8

Returns the extension of the file name (if any). This function will search for the file extension (separated by a .) and will return the text after the .. Files that end with ., or that start with . and have no other . in their name, are considered to have no extension. Examples:

  • "main.zig" ⇒ ".zig"
  • "src/main.zig" ⇒ ".zig"
  • ".gitignore" ⇒ ""
  • ".image.png" ⇒ ".png"
  • "keep." ⇒ "."
  • "src.keep.me" ⇒ ".me"
  • "/src/keep.me" ⇒ ".me"
  • "/src/keep.me/" ⇒ ".me"

The returned slice is guaranteed to have its pointer within the start and end pointer address range of path, even if it is length zero.
