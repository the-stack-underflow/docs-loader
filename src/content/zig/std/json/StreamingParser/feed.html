feed(
    p: *StreamingParser, 
    c: u8, 
    token1: *?Token, 
    token2: *?Token,
) Error!void

Give another byte to the parser and obtain any new tokens. This may (rarely) return two tokens. token2 is always null if token1 is null.

There is currently no error recovery on a bad stream.

Parameters

p: *StreamingParser,
c: u8,
token1: *?Token,
