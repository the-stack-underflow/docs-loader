resize(
    self: *@This(), 
    allocator: Allocator, 
    new_len: usize, 
    fill: bool,
) !void

Resizes to a new bit_length. If the new length is larger than the old length, fills any added bits with fill. If new_len is not zero, deinit must eventually be called.
