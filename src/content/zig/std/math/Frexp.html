frexp(
    x: anytype,
) Frexp(@TypeOf(x))

Breaks x into a normalized fraction and an integral power of two. f == frac 2^exp, with |frac| in the interval [0.5, 1).

Special Cases:

  • frexp(+-0) = +-0, 0
  • frexp(+-inf) = +-inf, 0
  • frexp(nan) = nan, undefined
