addWrap(
    r: *Managed, 
    a: *const Managed, 
    b: *const Managed, 
    signedness: Signedness, 
    bit_count: usize,
) Allocator.Error!bool

r = a + b with 2s-complement wrapping semantics. Returns whether any overflow occured.

r, a and b may be aliases.

Returns an error if memory could not be allocated.
