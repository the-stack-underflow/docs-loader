log1p(
    x: anytype,
) @TypeOf(x)

Returns the natural logarithm of 1 + x with greater accuracy when x is near zero.

Special Cases:

  • log1p(+inf) = +inf
  • log1p(+-0) = +-0
  • log1p(-1) = -inf
  • log1p(x) = nan if x < -1
  • log1p(nan) = nan
