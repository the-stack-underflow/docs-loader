log10(
    x: anytype,
) @TypeOf(x)

Returns the base-10 logarithm of x.

Special Cases:

  • log10(+inf) = +inf
  • log10(0) = -inf
  • log10(x) = nan if x < 0
  • log10(nan) = nan
