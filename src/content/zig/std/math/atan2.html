atan2(
    comptime T: type, 
    y: T, 
    x: T,
) T

Returns the arc-tangent of y/x.

Special Cases:

  • atan2(y, nan) = nan
  • atan2(nan, x) = nan
  • atan2(+0, x>=0) = +0
  • atan2(-0, x>=0) = -0
  • atan2(+0, x<=-0) = +pi
  • atan2(-0, x<=-0) = -pi
  • atan2(y>0, 0) = +pi/2
  • atan2(y<0, 0) = -pi/2
  • atan2(+inf, +inf) = +pi/4
  • atan2(-inf, +inf) = -pi/4
  • atan2(+inf, -inf) = 3pi/4
  • atan2(-inf, -inf) = -3pi/4
  • atan2(y, +inf) = 0
  • atan2(y>0, -inf) = +pi
  • atan2(y<0, -inf) = -pi
  • atan2(+inf, x) = +pi/2
  • atan2(-inf, x) = -pi/2

Parameters

T: type,
y: T,
x: T,
