split(
    comptime T: type, 
    buffer: []const T, 
    delimiter: []const T,
) SplitIterator(T)

Returns an iterator that iterates over the slices of buffer that are separated by bytes in delimiter.

split(u8, "abc|def||ghi", "|") will return slices for "abc", "def", "", "ghi", null, in that order.

If delimiter does not exist in buffer, the iterator will return buffer, null, in that order. The delimiter length must not be zero.

See also: tokenize and splitBackwards.
