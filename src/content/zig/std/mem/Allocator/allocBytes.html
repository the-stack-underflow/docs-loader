This function allows a runtime alignment value. Callers should generally prefer to call the alloc* functions.

Parameters

self: Allocator,
alignment: [1]anyopaque,

Must be >= 1. Must be a power of 2. Returned slice's pointer will have this alignment. 

byte_count: usize,
len_align: [1]anyopaque,

0 indicates the length of the slice returned MUST match byte_count exactly non-zero means the length of the returned slice must be aligned by len_align byte_count must be aligned by len_align
