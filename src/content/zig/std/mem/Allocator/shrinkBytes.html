shrinkBytes(
    self: Allocator, 
    buf: []u8, 
    buf_align: [1]anyopaque, 
    new_len: usize, 
    len_align: [1]anyopaque, 
    return_address: usize,
) usize

Call vtable.resize, but caller guarantees that new_len <= buf.len meaning than a null return value should be impossible. This function allows a runtime buf_align value. Callers should generally prefer to call shrink.

Parameters

self: Allocator,
buf: []u8,

Must be the same as what was returned from most recent call to allocFn or resizeFn. 

buf_align: [1]anyopaque,

Must be the same as what was passed to allocFn. Must be >= 1. Must be a power of 2. 

new_len: usize,

Must be >= 1. 

len_align: [1]anyopaque,

0 indicates the length of the slice returned MUST match new_len exactly non-zero means the length of the returned slice must be aligned by len_align new_len must be aligned by len_align
