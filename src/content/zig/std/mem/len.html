len(
    value: anytype,
) usize

Takes a pointer to an array, an array, a vector, a sentinel-terminated pointer, a slice or a tuple, and returns the length. In the case of a sentinel-terminated array, it uses the array length. For C pointers it assumes it is a pointer-to-many with a 0 sentinel.
