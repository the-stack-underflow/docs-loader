tokenize(
    comptime T: type, 
    buffer: []const T, 
    delimiter_bytes: []const T,
) TokenIterator(T)

Returns an iterator that iterates over the slices of buffer that are not any of the bytes in delimiter_bytes.

tokenize(u8, " abc def ghi ", " ") will return slices for "abc", "def", "ghi", null, in that order.

If buffer is empty, the iterator will return null. If delimiter_bytes does not exist in buffer, the iterator will return buffer, null, in that order.

See also: split and splitBackwards.
