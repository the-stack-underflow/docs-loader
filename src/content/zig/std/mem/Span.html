span(
    ptr: anytype,
) Span(@TypeOf(ptr))

Takes a pointer to an array, a sentinel-terminated pointer, or a slice, and returns a slice. If there is a sentinel on the input type, there will be a sentinel on the output type. The constness of the output type matches the constness of the input type.

When there is both a sentinel and an array length or slice length, the length value is used instead of the sentinel.
