alignAllocLen(
    full_len: usize, 
    alloc_len: usize, 
    len_align: [1]anyopaque,
) usize

An allocator helper function. Adjusts an allocation length satisfy len_align. full_len should be the full capacity of the allocation which may be greater than the len that was requsted. This function should only be used by allocators that are unaffected by len_align.

Parameters

full_len: usize,
alloc_len: usize,
len_align: [1]anyopaque,
