The linkedit_data_command contains the offsets and sizes of a blob of data in the LINKEDIT segment.

Fields

cmd: LC,

LC_CODE_SIGNATURE, LC_SEGMENT_SPLIT_INFO, LC_FUNCTION_STARTS, LC_DATA_IN_CODE, LC_DYLIB_CODE_SIGN_DRS or LC_LINKER_OPTIMIZATION_HINT. 

cmdsize: u32,

sizeof(struct linkedit_data_command) 

dataoff: u32,

file offset of data in LINKEDIT segment 

datasize: u32,

file size of data in LINKEDIT segment
