A map from keys to values, using an index lookup. Uses a bitfield to track presence and a dense array of values. This type does no allocation and can be copied by value.

Fields

bits: BitSet,

Bits determining whether items are in the map 

values: [Indexer.count]Value,

Values of items in the map. If the associated bit is zero, the value is undefined.
