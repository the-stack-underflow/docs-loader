assumeSentinel(
    p: anytype, 
    comptime sentinel_val: Elem(@TypeOf(p)),
) Sentinel(@TypeOf(p), sentinel_val)

Takes a Slice or Many Pointer and returns it with the Type modified to have the given sentinel value. This function assumes the caller has verified the memory contains the sentinel value.
