isTag(
    tagged_value: anytype, 
    comptime tag_name: []const u8,
) bool

Given an enum or tagged union, returns true if the comptime-supplied string matches the name of the tag value. This match process should be, at runtime, O(1) in the number of tags available to the enum or union, and it should also be O(1) in the length of the comptime tag names.
